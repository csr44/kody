Matlab project deals with optimization of parameters: rezistivity, inductance and capacitance of equivalent models of passive components used in EMI (electromagnetic interference) filters, such as resistor, coil and capacitor for certain frequency range. Three algorythms were used: Genetic algorythm linear and logarithmic, Particle Swarm. For successful optimization is needed file for configuration of optimization, "cost" function witch provide suitability of a particular solution and impedance function to recalculate the substituted values (R, L, C, fr.).
